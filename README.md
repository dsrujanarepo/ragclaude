# Python Projects Generic

A collection of Python projects and notebooks covering NLP, text similarity, word embeddings, and RAG (Retrieval-Augmented Generation).

## Contents

### Text Similarity & String Matching
- `StringSimilarities.ipynb` — Overview of string similarity techniques
- `Levenshtein.ipynb` — Levenshtein edit distance
- `Rapidfuzz.ipynb` — Fast fuzzy string matching with RapidFuzz
- `Cntvec_CosineSim.ipynb` — Cosine similarity using Count Vectorizer
- `TDIDF_CosineSim.ipynb` — Cosine similarity using TF-IDF

### Bag of Words (BoW)
- `E_BoW_CountVectorizer.ipynb` — BoW with Count Vectorizer
- `E_BoW_Tf_idf_Vectorizer.ipynb` — BoW with TF-IDF Vectorizer

### Word Embeddings
- `Word_Embeddings.ipynb` — Introduction to word embeddings
- `E_Neural_Word2Vec_Cbow.ipynb` — Word2Vec CBOW from scratch
- `E_Neural_Word2Vec_Cbow_Google.ipynb` — Word2Vec with Google's pre-trained model
- `E_Neural_GloVe.ipynb` — GloVe embeddings
- `E_Neural_FastText.ipynb` — FastText embeddings
- `E_Neural_BERT.ipynb` — BERT embeddings
- `E_Neural_GPT.ipynb` — GPT embeddings
- `E_Neural_Multimodal.ipynb` — Multimodal embeddings

### RAG Project
- `RAGClaudeCode/` — A Retrieval-Augmented Generation application built with Claude

### Scripts
- `asyncdemo.py` — Async Python demo
- `Leidos.py` / `LeidosPlaywright.py` — Playwright-based web automation scripts

### Other
- `OLD/` — Archived scripts and notebooks
